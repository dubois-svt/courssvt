## Séance 1 : Chaine d'énergie

!!! question "Problématique"

    Comment représenter l'énergie dans les OST ?

[Activité La chaine d'énergie](../chaineEnergie)




??? abstract "Bilan"

    <a markdown id="bilan1">

    L'énergie est indispensable pour réaliser des action. Elle se présente sous différentes formes, comme l'énergie électrique, cinétique, potentielle, thermique et lumineuse.

    Une chaine d'énergie représente le chemin parcouru par l’énergie dans un système technique. A chaque bloc de la chaine est associé une solution technique de l'OST.

    Dans une chaine d'énergie, il y a différents éléments qui servent à:

    - alimenter (batterie),
    - distribuer (relais, interrupteur),
    - convertir (moteur, lampe, radiateur, génératrice, vérin) et
    - transmettre (engrenages, courroies, chaine)

    Les objets techniques transforment une forme d'énergie en une autre pour fonctionner. 
    Cette conversion s'accompagne souvent de pertes d'énergie sous forme de chaleur.




    </a>

## Séance 2 : Chaine d'information

!!! question "Problématique"

    Comment représenter l'information dans les OST ?

[Activité La chaine d'information](../chaineInfo)




??? abstract "Bilan"

    <a markdown id="bilan2">

    Une chaine d'information représente le chemin parcouru par l’information dans un système technique automatisé. A chaque bloc de la chaine est associé une solution technique de l'OST.


    Dans une chaine d'information, il y a différents éléments qui servent à:

    - Acquérir (capteurs), 
    - Traiter (microcontrôleur ou microprocesseur) et
    - Communiquer à des actionneurs (moteur ...) ou humain

    Les chaines d'énergie et d'informations permettent de comprendre le fonctionnement d'un OST automatisé.

    </a>
